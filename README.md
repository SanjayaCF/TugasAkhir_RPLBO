# MembershipMaster

## Penjelasan Singkat Aplikasi Prototype
MembershipMaster adalah sebuah aplikasi prototype yang dirancang untuk mengelola keanggotaan (membership) sebuah layanan atau platform. Aplikasi ini memungkinkan pengguna untuk melakukan registrasi, login, mengelola langganan membership, mencari daftar membership, menampilkan informasi detail membership, melakukan pembatalan langganan, serta memberikan pengingat perpanjangan langganan. Selain itu, aplikasi ini juga dapat menampilkan total pengeluaran pengguna dan melakukan analisis pengeluaran pengguna.
